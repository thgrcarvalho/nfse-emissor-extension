Preencher NFS-e

A extensão precisa de permissão de acesso ao Emissor Nacional (e ao Banco Central, para buscar o câmbio).

O Firefox não concede essa permissão automaticamente ao carregar a extensão — clique abaixo e aceite. Depois disso, a página do portal será recarregada.

Esta extensão funciona apenas no Emissor Nacional (www.nfse.gov.br/EmissorNacional).

Abra o portal e faça login para usar.

Você precisa estar logado no Emissor Nacional para usar a extensão.

Faça login com CPF/CNPJ e senha — o painel libera o preenchimento automaticamente.

Não consegui ler a página do portal.

Recarregue a aba e tente novamente — depois de instalar ou atualizar a extensão é preciso recarregar a página do portal.

Não consegui identificar o CNPJ logado.

Por segurança, o preenchimento fica bloqueado sem identificar o cliente. Recarregue a página; se o problema continuar, o portal pode ter mudado e a extensão precisa de atualização.

Você está logado. Inicie a Emissão completa para preencher com os dados acima.

Para usar ou trocar o cliente de referência, abra uma nota já emitida deste cliente (no portal) e escolha Usar nesta emissão ou Salvar como padrão.

Clientes salvos neste navegador

Excluir remove os dados salvos do cliente deste navegador (clique duas vezes para confirmar). O perfil pode ser recriado a partir de uma nota emitida.

Revise os dados da nota e clique em Emitir NFS-e no final da página.

A extensão não emite por você — a emissão é sempre manual.

Ainda não há um perfil para o CNPJ logado.

Abra uma nota já emitida deste cliente (no portal) e escolha Salvar como padrão para criar o perfil — assim o preenchimento usa sempre os dados corretos.
